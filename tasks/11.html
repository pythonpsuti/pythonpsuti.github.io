Краткое описание:

* Продемонстрировать работу метода

Код программы:

import math

def z(x, y):
    return (x + (2+y)/(x ** 2)) / (y + 1/( math.sqrt(x ** 2 + 10 ) ))

def q(x, y):
    return 2.8 * math.sin(x) + abs(y)

x = float( input('Введите значение x: ') )
y = float( input('Введите значение y: ') )
print('z(x,y) = ', z(x,y))
print('q(x,y) = ', q(x,y))
