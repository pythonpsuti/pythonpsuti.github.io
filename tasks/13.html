Задание №13

Напишите программу для моделирования бросания игрального кубика каждым из двух игроков. Определить, кто из игроков получил на кубике больше очков.


Краткое описание:
* Смоделировать бросок кубика двух игроков
* Сравнить значения и объявить победителя

Вспомогательная информация:

Для того, чтобы смоделировать бросок именно игроком, воспользуемся методом input(), однако в этот раз не будем учитывать то, что ввёл пользователь (вызов без присвоения). Здесь важен сам факт приостановки скрипта до нажатия пользователем клавиши подтверждения ввода (ENTER).

Создание случайного числа в диапазоне от a до b происходит при помощи метода randint(a, b) из модуля random, который нужно предварительно импортировать


Код программы:

import random

input('Ожидание хода игрока 1... (Нажмите ENTER)')
a = random.randint(1,6)
print('Выпало очков -', a)

input('Ожидание хода игрока 2... (Нажмите ENTER)')
b = random.randint(1,6)
print('Выпало очков -', b)

if a > b:
    print('Победил игрок 1')
elif a < b:
    print('Победил игрок 2')
else:
    print('Ничья')
