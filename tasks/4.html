Задание №4

Напишите программу, проверяющую целое число на четность. Реализовать в виде вызова собственной функции. 

Краткое описание:
* Запросить с клавиатуры ввод целого числа
* Написать метод для проверки целого числа на чётность
* Продемонстрировать работу метода

Вспомогательная информация:

Для проверки числа на чётность будем использовать оператор % - возвращает остаток от целочисленного деления числа слева на число справа.
    Например, 10 % 4 = 2 или 101 % 10 = 1
Алгоритм проверки: если остаток от деления нашего числа на 2 равен единице, значит число НЕчётное и наоборот.


Код программы:

def checkParity(num):
    if num % 2 == 0:
        return True
    else:
        return False

a = int( input('Введите целое число: ') )
if checkParity(a) == True:
    print('Число чётное')
else:
    print('Число нечётное')
