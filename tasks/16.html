Задание №16

L = [3, 6, 7, 4, -5, 4, 3, -1] 
1. Определите сумму элементов списка L. ЕСЛИ сумма превышает значение 2, ТО вывести на экран число элементов списка. 

2. Определить разность между минимальным и максимальным элементами списка. ЕСЛИ абсолютное значение разности больше 10, ТО вывести на экран отсортированный по возрастанию список, ИНАЧЕ вывести на экран фразу «Разность меньше 10». 

Краткое описание:
* Обращаем внимание на задание, про модуль не сказано, значит всё делается в одном файле
* Продемонстрировать работу

Вспомогательная информация:

Количество элементов в списке - это длина списка. Находится так же, как и длина строки - через метод len().

Сумма элементов списка, состоящего из чисел, находится при помощи метода sum().

Минимальное и максимальное значение числа, находящегося в списке, состоящем из чисел, находится при помощи методов min() и max() соответственно.

Абсолютное значение находится при помощи метода abs().

Для того чтобы отсортировать числовые значения по возрастанию, нужно использовать метод sorted, принимающий в качестве аргумента сам список и ключ. Важно указать, чтобы ключ сортировал значение именно как числа (int).


Код программы:

L = [3, 6, 7, 4, -5, 4, 3, -1] 

if sum(L) > 2:
    print( len(L) )
    
if abs(min(L) - max(L)) > 10:
    print( sorted(L, key=int) )
else:
    print('Разность меньше 10')
