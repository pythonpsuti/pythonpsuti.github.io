Задание №18

L = [3, 'hello', 7, 4, 'привет', 4, 3, -1]
Определите наличие строки «привет» в списке. ЕСЛИ такая строка в списке присутствует, ТО удалить ее из списка, ИНАЧЕ добавить строку в список. 
Подсчитать, сколько раз в списке встречается число 4, ЕСЛИ больше одного раза, ТО очистить список.

Краткое описание:
* Выполняем поиск строки и удаляем, если есть или добавляем, если нет
* Считаем сколько цифр 4 в списке и, если больше 1, то чистим список

Вспомогательная информация:

Удаление элемента из списка по значению позволяет выполнить метод remove(), применяемый к списку

Добавление элемента в конец списка позволяет выполнить метод append(), применяемый к списку

Наличие элемента в списке можно определить при помощи вспомогательного слова in
    Например, if 'abc' in СПИСОК:
    Вернёт True, если 'abc' есть в списке СПИСОК, иначе False


Код программы:

L = [3, 'hello', 7, 4, 'привет', 4, 3, -1]

if 'привет' in L:
    L.remove('привет')
else:
    L.append('привет')
print(L)

if L.count(4) > 1:
    L.clear()
print(L)
