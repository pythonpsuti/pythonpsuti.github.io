Задание №5

Напишите программу, вычисляющую значение функции (на вход подается вещественное число):


Краткое описание:
* Запросить ввод с клавиатуры вещественного числа
* Написать метод в соответствии с данной функцией
* Продемонстрировать работу метода

Вспомогательная информация:

Для возведения числа в квадрат используется оператор **
    a ** b = a * ... * a (b раз)
    Например, 2 ** 2 = 4 или 3 ** 2 = 9


Код программы:

def func(x):
    if x >= -2.4 and x <= 5.7:
        return x ** 2
    else:
        return 4

a = float( input('Введите число: ') )
print( 'Результат работы функции с данным числом =', func(a) )
