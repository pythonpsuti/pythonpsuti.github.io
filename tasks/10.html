Задание №10

Напишите программу-игру в виде отдельного модуля. Компьютер загадывает случайное число, пользователь пытается его угадать. Программа запрашивает число ОДИН раз. Если число угадано, то выводим на экран «Победа», иначе – «Повторите еще раз». Для написания программы понадобится функция randint() из модуля random.

Краткое описание:
* Написать модуль, который генерирует случайное число и запрашивает ввод с клавиатуры у пользователя. После этого происходит сравнение чисел  сгенерированного и введённого пользователем и, в случае совпадения - сообщаем о победе, иначе - проигрыш.
* По условию следует запускать игру из другого модуля

Код программы:

Основной файл zadanie10.py

import numberGame

numberGame.play()

Модуль игры numberGame.py
import random

def play():
    a = random.randint(1, 5)
    b = int( input('Я загадал число от 1 до 5. Ваш ответ: ') )
    if a == b:
        print('Победа')
    else:
        print('Повторите ещё раз')
