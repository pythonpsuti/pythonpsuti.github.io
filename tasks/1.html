Задание №1

Напишите программу, определяющую сумму и произведение трех чисел (типа int, float), введенных с клавиатуры. 
    
    Пример работы программы:
    Введите первое число: 1 
    Введите второе число: 4 
    Введите третье число: 7
    Сумма введенных чисел: 12 
    Произведение введенных чисел: 28
    

Краткое описание:
* Запрашиваем ввод 3-х чисел в переменные a, b, c
* Преобразовываем данные (input всегда возвращает строку) в числовой тип данных
* Производим нужные действия: сложение и умножение

Подробное описание:
1. Требуется запросить ввод данных от пользователя в виде трёх чисел типа int (целоцисленное) или float (с плавающей точкой/дробное) введённых с клавиатуры.
Согласно Python, float приоритетнее int, поэтому преобразовывать полученные данные мы будем именно в float. 


text = input('Введите что-нибудь...'); # Метод запроса на ввод данных с клавиатуры
АРГУМЕНТЫ МЕТОДА:
- текст, который выводится до запроса ввода и информирует пользователя о том, что нужно ввести.
ВОЗВРАЩАЕТ: возвращает ВСЕГДА строку c введёнными с клавиатуры данными.

num1 = int('12'); # Метод преобразования в число с плавающей точкой
num2 = float('12.5'); # Метод преобразования в число с плавающей точкой
АРГУМЕНТЫ МЕТОДА:
- текст (строка), состоящий из цифр
ВОЗВРАЩАЕТ: возвращает числовое значение в зависимости от выбранного метода


2. Второй и заключительный шаг - вывод информации, требуемой в задании. Для этого следует воспользоваться методом print().


print('Введите что-нибудь...'); # Метод вывода текста в консоль
АРГУМЕНТЫ МЕТОДА:
- текст, который будет выведен в консоль


Код программы:

a = float( input('Введите первое число: ') )
b = float( input('Введите второе число: ') )
c = float( input('Введите третье число: ') )

print('Сумма введённых чисел: ', a+b+c)
print('Произведение введённых чисел: ', a*b*c)
