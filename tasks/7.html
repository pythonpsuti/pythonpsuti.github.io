Задание №7

Найдите площадь треугольника с помощью формулы Герона. Стороны задаются с клавиатуры. Реализовать вычисление площади в виде функции, на вход которой подаются три числа, на выходе – площадь. Функция находится в отдельном модуле, где происходит разделение между запуском и импортированием. Описание математических функций можно найти в документации


Краткое описание:
* Написать модуль для вычисления площади по формуле Герона и добавить различие: модуль импортирован или запущен отдельно
* Продемонстрировать работу метода

Вспомогательная информация:

Для того, чтобы выполнить действия в модуле, только если он был запущен отдельно, нужно использовать следующее условие:
    if __name__ == "__main__":
        набор действий, выполняемых если модуль запущен отдельно..

Модуль geronModule.py

import math

def geron(a, b, c):
    p = (a + b + c) / 2
    S = math.sqrt( p * (p-a) * (p-b) * (p-c) )
    return S

if __name__ == "__main__":
    a = float( input('Введите первую сторону: ') )
    b = float( input('Введите вторую сторону: ') )
    c = float( input('Введите третью сторону: ') )
    print( 'Площадь треугольника =', geron(a,b,c) )
