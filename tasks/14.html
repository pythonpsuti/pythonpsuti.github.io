Задание №14

s = "У лукоморья 123 дуб зеленый 456" 
1. Определить, встречается ли в строке буква 'я'. Вывести на экран ее позицию (индекс) в строке. 
2. Определить, сколько раз в строке встречается буква 'у'. 
3. Определить, состоит ли строка только из букв, ЕСЛИ нет, ТО вывести строку в верхнем регистре. 
4. Определить длину строки. ЕСЛИ длина строки превышает 4 символа, ТО вывести строку в нижнем регистре. 
5. Заменить в строке первый символ на 'О'. Результат вывести на экран.

Краткое описание:
* Выполняем задания попунктно, см. вспомогательную информацию для помощи

Вспомогательная информация:

Для того, чтобы определить встречается ли буква в строке, и если есть, получить её индекс в строке, можно воспользоваться методом find, применяемом к строке.
    Например, 'привет'.find('е') вернёт 4
        или 'привет'.find('a') вернёт -1

Количество повторений любой последовательности символов (в нашем случае буквы) в строке, используется метод count(), применяемый к строке. Аргументом является то, на что идёт проверка.
    Например, 'abcdabcdabcf'.count('abcd') вернёт 2
        или '1231231234123'.count('1') вернёт 4
        
Проверка на то, состоит ли строка только из букв используется метод isalpha(), применяемый к строке.
    Например, 'abcdпривет'.isalpha() вернёт True
        или 'abc123test'.isalpha() вернёт False
        
Перевод строки в верхний регистр осуществляет метод upper(), в нижний - lower(), применяется к строке.
    Например, 'abcdпривет'.upper() вернёт 'ABCDПРИВЕТ'
    
Длину строки определяет метод len().
    Например, len('abcdпривет') вернёт 10
    
Для замены первого символа воспользуемся следующим методом: в эту же переменную присвоим нужную букву (например, "О") и здесь же прибавим исходную строку, НО начиная со второго символа.
    stroka = 'Проверка'
    stroka = 'O' + stroka[1:]
    stroka -> 'Ороверка'


Код программы:

s = "У лукоморья 123 дуб зеленый 456"

a = s.find('я')
if a != -1:
    print('Индекс буквы "я" в строке =', a)
    
b = s.count('у')
print('Количество букв "у" в строке =', b)

if s.isalpha() == False:
    print( s.upper() )

if len(s) > 4:
    print( s.lower() )

s = 'О' + s[1:]
print(s)
