Задание №12

Напишите программу, вычисляющую значение функции (на вход подается вещественное число):


Краткое описание:
* Написать метод для определения значения функции
* Продемонстрировать работу метода

Код программы:

import math

def func(x):
    if x >= 0.2 and x <= 0.9:
        return math.sin(x)
    else:
        return 1

a = float( input('Введите число: ') )
print( 'Результат работы функции с данным числом =', func(a) )
