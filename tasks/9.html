Создайте в отдельном модуле функцию для вычисления выражения:


Краткое описание:
* Написать метод в соответствии с данным выражением
* Продемонстрировать работу метода

Код программы:

import math

def func(x):
    return math.sqrt( 1 - math.sin(x) ** 2 )

a = float( input('Введите x: ') )
print( 'Значение выражение в этой точке = ', func(a) )
