Задание №3

Напишите собственную программу, определяющую максимальное из двух введенных чисел. Реализовать в виде вызова собственной функции, возвращающей большее из двух переданных ей чисел.

Краткое описание:
* Запросить на ввод 2 числа с клавиатуры
* Написать метод для поиска и возврата наибольшего числа
* Продемонстрировать работу метода

Код программы:

def getMax(num1, num2):
    if num1 > num2:
        return num1
    else:
        return num2

a = int( input('Введите первое число: ') )
b = int( input('Введите второе число: ') )

print( 'Наибольшее из двух чисел = ', getMax(a, b) )
